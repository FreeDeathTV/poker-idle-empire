V1301 – Public Demo Mode (Playable Preview Build)
Status: Approved
Scope: Locked
Drift: Not permitted
Version: 1301
Feature: Public Demo Mode (/game) + Save‑Locked Preview + Install Conversion Funnel
Purpose: Provide a fully playable, non‑saving demo version of Poker Idle Empire accessible from the landing page. Unlock all minigames for preview. Convert players into PWA installs.

1. Overview
V1301 introduces a public, save‑locked demo version of Poker Idle Empire accessible at:

Code
/game
This route provides:

A fully playable preview of the core idle loop

All minigames unlocked for showcase

No saving of progress

A persistent “Install to Save” conversion funnel

An exit warning modal

A deterministic mode flag (demoMode = true)

This version is not the real game. It is a marketing preview designed to convert curious visitors into PWA users.
